# Agriculture Dashboard

A  agriculture management dashboard built with HTML, CSS, Bootstrap, and JavaScript.

## Features
- Dashboard with 4 stat cards
- Charts (Bar, Pie, Line)
- Crops table with CRUD operations
- Responsive design
- Sidebar navigation

## How to Run
Open index.html in any web browser
